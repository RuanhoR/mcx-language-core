# mcx-language-core README

# How to use this DSL?

The mcx DSL git repo is in github.com/RuanhoR/mcx-core  
If you want to create or build mcx project, run `npm i -g mbler` and `mbler c .` to point mbler work dir to current dir, run `mbler init` to create your project use template, run `mbler build` to build project, `mbler watch` to watch file change and build.
